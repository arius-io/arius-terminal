## Arius Terminal

Simple QML-based Terminal Emulator

### Features
* Lightweight
* Mobile UI friendly

### Dependencies
* [Qt 5.11](https://github.com/arius-io/arius-shell/blob/master/download.qt.io/official_releases/qt/5.11)
* [qmltermwidget](https://github.com/Swordfish90/qmltermwidget)

### Build instructions

```sh
$ mkdir build
$ cd build
$ qmake ../src
$ make
```
